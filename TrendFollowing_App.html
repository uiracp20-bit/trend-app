📡 Indicators

Hilo Activator
BUY if Close > SMA(P) of Highs
SELL if Close < SMA(P) of Lows
RSI (14-period)
Must be 30–70. >70 = overbought, <30 = oversold → no trade
Net Volume
Sum vol last 5 days vs prior 5. BULLISH if higher, BEARISH if lower.

🚦 Signal Rules

BUY (enter long)
Hilo = BUY + Net Vol = BULLISH + RSI 30–70 ✅
SELL (close position)
Hilo = SELL + Net Vol = BEARISH + RSI 30–70
WATCH
Partial confirmation only — do not act

💼 Risk Management

Risk per trade
Configurable % of capital (default 2%)
Stop Loss
= Hilo Value (HiloLow when BUY, HiloHigh when SELL)
Position Size
Risk $ ÷ (Entry − Stop) = number of shares
Trailing Stop
Hilo value rises with the trend — raise your stop to the latest Hilo each session (see Positions tab)
Platform note
Stake is long-only. SELL = close trade, not short.
